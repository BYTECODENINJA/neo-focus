# NEO FOCUS

![NEO FOCUS](neo-focus.ico)

NEO FOCUS is a comprehensive desktop productivity application designed to help you manage tasks, build habits, track goals, and maintain focus. Built with modern technologies and featuring a beautiful, intuitive interface with advanced analytics and achievement systems.

## ✨ Features

### 🎯 Core Productivity Tools
- **📅 Smart Calendar**: Drag-and-drop scheduling with task and habit integration
- **✅ Task Management**: Priority-based task organization with flexible scheduling
- **🎯 Habit Tracking**: Custom schedules (daily, weekly, specific days) with streak tracking
- **🏆 Goal Setting**: Visual progress tracking with milestone celebrations
- **📝 Rich Notes**: Markdown support with tags and advanced formatting
- **📔 Enhanced Journaling**: Mood tracking, energy levels, and reflection prompts

### 📊 Analytics & Insights
- **📈 Progress Visualization**: Charts and graphs for all productivity metrics
- **🎯 Goal Progress Charts**: Visual representation of goal completion
- **📊 Habit Streak Analytics**: Track consistency and longest streaks
- **📈 Productivity Trends**: Weekly, monthly, and yearly progress analysis
- **🧠 Mood & Energy Tracking**: Visualize emotional patterns over time

### 🏆 Achievements & Motivation
- **🎖️ Achievement System**: Unlock badges for consistency and milestones
- **🔥 Streak Tracking**: Visual streak counters with celebration effects
- **🎉 Celebration Animations**: Confetti and animations for completed goals
- **🏅 Milestone Rewards**: Special recognition for major achievements

### 📱 Smart Features
- **🔔 Custom Reminders**: Flexible notification system for tasks, habits, and breaks
- **🖱️ Drag-and-Drop Calendar**: Intuitive scheduling interface
- **📱 Desktop Notifications**: Native system notifications
- **🌙 Dark/Light Mode**: Seamless theme switching
- **💾 Offline-First**: Full functionality without internet connection

### 🔄 Review & Reflection
- **📋 Daily Reviews**: Guided reflection prompts for daily planning
- **📊 Weekly Reviews**: Comprehensive weekly progress analysis
- **🎯 Goal Reflection**: Regular check-ins on goal progress
- **📝 Gratitude Prompts**: Built-in gratitude and reflection exercises

## 🖥️ Desktop Application

NEO FOCUS is built as a native desktop application using Next.js and Electron, providing:

- **🖥️ Native Performance**: Runs smoothly on Windows, macOS, and Linux
- **💾 Local Database**: Local storage for reliable data persistence
- **🔄 Auto-Save**: Automatic data backup every 2 seconds
- **📱 System Integration**: Native menus, notifications, and shortcuts
- **🔒 Privacy-First**: All data stored locally on your device

## 🚀 Installation

### Download Pre-built Releases
1. Visit the [Releases](https://github.com/BYTECODENINJA/neo-focus/releases) page
2. Download the appropriate version for your operating system:
   - **Windows**: `NEO-FOCUS-Setup.exe`
   - **macOS**: `NEO-FOCUS.dmg`
   - **Linux**: `NEO-FOCUS.AppImage`
3. Install and run the application

### Build from Source

#### Prerequisites
- Node.js 18+ 
- npm or yarn

#### Installation Steps
\`\`\`bash
# Clone the repository
git clone https://github.com/yourusername/neo-focus.git
cd neo-focus

# Install dependencies
npm install

# Development mode (runs both Next.js and Electron)
npm run electron-dev

# Build for production
npm run build-electron

# Create distributable packages
npm run dist
\`\`\`

## ⚙️ Configuration

### Database Location
- **Windows**: `%APPDATA%/neo-focus/neo-focus.db`
- **macOS**: `~/Library/Application Support/neo-focus/neo-focus.db`
- **Linux**: `~/.config/neo-focus/neo-focus.db`

### Keyboard Shortcuts
- **Ctrl/Cmd + N**: Create new item
- **Ctrl/Cmd + S**: Save all data
- **Ctrl/Cmd + D**: Toggle dark mode
- **Ctrl/Cmd + A**: Open analytics
- **Ctrl/Cmd + R**: Daily review
- **Ctrl/Cmd + W**: Weekly review
